Join Minevictus 
on Java Edition

Would you like to join a Minecraft survival server which 
aims to bring teamwork and strategy back?

1) In Minecraft, go to Multiplayer and click on Add server.
2) Then enter play.minevict.us in the address box.
3) Click on Done and join the server with the latest version.

Make sure to also visit our Discord and Forum by using the buttons below to get a taste of our community. We're always happy to answer your questions. See you soon, adventurer.
